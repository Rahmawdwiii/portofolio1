Pengenalan HTML

HTML adalah bahasa pemrograman dalam membangun website. HTML merupakan singkatan dari Hypertext Markup Language. Ada berbagai macam tag elemen yang dapat digunakan seperti heading dan paragraf.

Dalam membuat halaman web,HTML dapat dikolaborasikan dengan bahasa lainnya seperti JavaScript dan CSS (Cascading Style Sheet). CSS adalah bahasa khusus yang digunakan untuk mempermudah tampilan web.
